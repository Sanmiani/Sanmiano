# Canva MCP — Reference Guide

**Status:** Connected via claude.ai MCP
**What it unlocks:** Generate, edit, and export designs without opening Canva manually.

---

## What you can do

| Action | How to ask |
|---|---|
| Generate a design | "Create an Instagram post design about [topic] using my brand colours" |
| Generate from a prompt | "Design a Facebook banner for African men's health week" |
| Search existing designs | "Find my heart health post designs in Canva" |
| Get design content | "Show me the text and layout of my latest Canva design" |
| Export a design | "Export my latest Instagram post as a PNG" |
| Upload an asset | "Upload this image to Canva for use in my designs" |
| List brand kits | "What brand colours and fonts do I have set up in Canva?" |

---

## Best use cases for your content workflow

1. **Weekly visual batch** — Ask Claude to generate 3 social media post designs (one per platform) from your content ideas
2. **Reels thumbnail** — Generate a thumbnail design for each Reel before posting
3. **Brand consistency** — Pull your brand kit before generating so designs match your colours/fonts
4. **Quote cards** — Turn a stat or tip from your caption into a shareable quote card

---

## Example prompts

```
Generate an Instagram post design for African men 35+ about high blood pressure. 
Use a bold, clean layout. Dark green and white colour scheme. 
Include the headline: "Your BP numbers matter. Here's what to do about it."
```

```
Create a Facebook Page cover image for a men's health community. 
Professional but warm. Include space for the community name.
```

```
Design a carousel template (5 slides) for an educational Instagram post. 
Slide 1 = hook/title. Slides 2-4 = tips. Slide 5 = CTA.
```

---

## Notes

- Canva MCP can create designs but cannot post directly to Instagram/Facebook — export first, then post
- Set up your brand kit in Canva (colours, fonts, logo) so generated designs stay on-brand
- Use `list-brand-kits` to confirm your kit is loaded before batch generating
