# Gmail MCP — Reference Guide

**Status:** Connected via claude.ai MCP (mattsanmiano@gmail.com)
**What it unlocks:** Search, read, draft, label, and organise emails without opening Gmail.

---

## What you can do

| Action | How to ask |
|---|---|
| Search emails | "Find any emails about brand partnerships or collaborations" |
| Read a thread | "Show me my last email from [person/brand]" |
| Create a draft | "Draft a reply to this email for me" |
| Label an email | "Label this email as 'Brand Deals'" |
| List labels | "What labels do I have set up in Gmail?" |
| Create a label | "Create a label called 'Community Members'" |

---

## Best use cases for your content workflow

1. **Brand deal tracking** — Search for inbound partnership emails and draft replies in your voice
2. **Community inbox** — Label emails from community members so nothing gets lost
3. **Newsletter drafts** — Ask Claude to draft email newsletters to your list, saved to Gmail drafts
4. **Research digests** — Search for health articles or AI news you've been sent and summarise them

## Example prompts

```
Search my Gmail for any emails about health content, partnerships, or sponsorships 
from the last 30 days. Summarise what's there.
```

```
Draft a welcome email to someone who just joined my Facebook community. 
Write it in my voice — warm, direct, brothers-first. Save it as a Gmail draft.
```

```
Create a Gmail label called "Content Ideas" and one called "Brand Deals" 
so I can start organising incoming opportunities.
```

## Notes

- Gmail MCP can draft but not send — you review and send from Gmail
- Use labels aggressively; your inbox is currently scattered
- Good first action: ask Claude to set up 5 labels for your workflow (Brand Deals, Community, Content Ideas, Partnerships, Admin)
