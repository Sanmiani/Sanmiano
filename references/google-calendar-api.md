# Google Calendar MCP — Reference Guide

**Status:** Connected via claude.ai MCP
**What it unlocks:** Read, create, and manage calendar events — including your content posting schedule.

---

## What you can do

| Action | How to ask |
|---|---|
| List events | "What's on my calendar this week?" |
| Create an event | "Add a reminder to post on Instagram every Monday, Wednesday, Friday at 7pm" |
| Get a specific event | "Show me what I have on Thursday" |
| Update an event | "Move my Friday post reminder to 8pm" |
| Delete an event | "Remove next Tuesday's reminder" |
| Suggest a time | "When's a good time this week for a 1-hour content batch session?" |

---

## Best use cases for your content workflow

1. **Posting reminders** — Create recurring events for your 3x/week posting schedule
2. **Content batch blocks** — Block 2-hour sessions for writing and scheduling content
3. **Weekly review** — Add a Friday 30-min block to review performance and plan next week
4. **Audit/level-up rituals** — Schedule your Day 7 /audit and Day 14 /level-up sessions now

---

## Example prompts

```
Create recurring calendar events for posting on Instagram, LinkedIn, and Facebook. 
Schedule them for Monday, Wednesday, and Friday at 7pm each week. 
Call them "Post: [Platform]". Start from this Monday.
```

```
Block 2 hours every Sunday at 10am for content batching. 
Call it "Content Batch Session". Recurring weekly.
```

```
Add a reminder for next Friday at 5pm: "Run /audit on AIOS".
```

---

## Notes

- Calendar MCP sees your existing Google Calendar — it knows when you're free
- Use it to protect your content creation time, not just track it
- Posting reminders work best at the same time each day — pick a time you're reliably free
