# Gamma MCP — Reference Guide

**Status:** Connected via claude.ai MCP
**What it unlocks:** Generate AI-powered presentations, documents, and web pages — fast.

---

## What you can do

| Action | How to ask |
|---|---|
| Generate a presentation | "Create a presentation about heart health for African men 35+" |
| Generate a document | "Create a one-pager on the top 5 health risks for Black men" |
| Generate a webpage | "Create a landing page for my men's health community" |
| Read an existing Gamma | "Show me the content of my last Gamma presentation" |
| List your Gammas | "What presentations do I have in Gamma?" |
| Get themes | "What design themes are available in Gamma?" |

---

## Best use cases for your content workflow

1. **Lead magnets** — Generate a "5 Health Checks Every African Man Over 35 Should Do" PDF/doc to grow your email list
2. **Community presentations** — Create slide decks for live sessions or webinars in your community
3. **Website landing page** — Gamma can generate a basic web page — useful while your full site is in progress
4. **Content upgrades** — Turn a popular post into a detailed downloadable guide

---

## Example prompts

```
Generate a presentation called "The African Man's Guide to Heart Health". 
5 slides. Audience: African men 35+. Tone: direct, warm, evidence-based.
Sections: Know Your Numbers / Eat Right / Move Daily / Manage Stress / Sleep Better.
```

```
Create a one-page document: "5 Blood Pressure Facts Every Brother Over 35 Must Know".
Formatted as a downloadable resource. Short paragraphs. Bold key stats.
```

```
Generate a simple landing page for a community called "[Community Name]" — 
for African men 35+ focused on health and AI. Include a headline, 3 benefits, and a join button.
```

---

## Notes

- Gamma generates the content but you customise it in the Gamma editor — Claude cannot edit existing Gammas
- Use Gamma for lead magnets and community resources, not for daily social posts
- Generated Gammas can be shared as links — useful for adding to your bio or pinned posts
