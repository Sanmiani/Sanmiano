# Brave Search MCP — Reference Guide

**Status:** Configured in `.claude/settings.json` — needs API key to activate
**What it unlocks:** Real-time web search for research, health stats, AI news — directly in Claude.

---

## How to activate

1. Go to [brave.com/search/api](https://brave.com/search/api/)
2. Sign up for a free account (free tier = 2,000 searches/month — enough for your use)
3. Copy your API key
4. Open `.claude/settings.json` and replace `YOUR_BRAVE_API_KEY_HERE` with your key
5. Restart Claude Code — the search tool will be live

---

## Best use cases for your content workflow

1. **Stat hunting** — Find the specific numbers that make your posts credible
2. **Weekly content research** — Pull 3-5 fresh health facts every Monday for the week's posts
3. **AI news digest** — Search for AI tools or stories relevant to your community
4. **Competitor/inspiration research** — Find what top men's health accounts are posting about
