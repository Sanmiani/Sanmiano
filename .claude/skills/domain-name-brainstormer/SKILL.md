# Domain Name Brainstormer

This skill helps find the perfect domain name by generating creative options and checking what's actually available to register.

## When to Use This Skill

- Starting a new project or company
- Launching a product or service
- Creating a personal brand or portfolio site
- Rebranding an existing project
- Finding available alternatives when your first choice is taken

## What This Skill Does

1. **Understands Your Project**: Analyzes what you're building and who it's for
2. **Generates Creative Names**: Creates relevant, memorable domain options
3. **Checks Availability**: Verifies which domains are available across multiple TLDs
4. **Multiple Extensions**: Suggests .com, .io, .dev, .ai, .app, and more
5. **Branding Insights**: Explains why certain names work well

## How to Use

```
I'm building [project description] for [audience]. Suggest domain names.
```

```
I need a domain for my personal brand as a [role]. Prefer short names.
```

```
Suggest domain names using the words "[keyword]" for my [type of project].
```

## Instructions

### Step 1: Understand the Project

Ask if not provided:
- What does the project do?
- Who is the audience?
- Any keywords to include or avoid?
- TLD preference (.com, .io, .ai, etc.)?
- Style preference? (descriptive, brandable, personal name, acronym)

### Step 2: Generate Name Options

Create 10-15 options across categories:

- **Descriptive**: Clearly states what it does
- **Brandable**: Unique, made-up word or creative combo
- **Action-based**: Starts with a verb
- **Audience-based**: References the target person
- **Keyword combo**: Two relevant words combined

### Step 3: Output Format

```
🎯 Domain Name Suggestions

## Available (.com)
1. ✓ [name].com — [Why it works]
2. ✓ [name].com — [Why it works]

## Available (Alternative TLDs)
3. ✓ [name].io — [Why it works]
4. ✓ [name].ai — [Why it works]

## Premium/Taken
- [name].com (Taken, est. $X,XXX)

## Top Recommendations

🏆 Top Pick: [name].[tld]
- [Why this is the best choice]
- [Availability status]

🥈 Runner-up: [name].[tld]
- [Why this is second best]
```

### Step 4: Branding Notes

For each top pick, note:
- Pronounceability (can it be said in conversation?)
- Memorability (will people spell it correctly?)
- Brandability (does it stand out?)
- Longevity (will it still make sense in 5 years?)

## What Makes a Good Domain

| Quality | Target |
|---|---|
| Length | Under 15 characters ideal |
| Spelling | Easy to spell from hearing it |
| Pronunciation | Can be said out loud |
| Hyphens | Avoid — hard to share verbally |
| Numbers | Avoid — ambiguous verbally |

## TLD Guide

| TLD | Best for |
|---|---|
| .com | Universal, trusted, great for businesses |
| .io | Tech startups, developer tools |
| .dev | Developer-focused products |
| .ai | AI/ML products |
| .app | Mobile or web applications |
| .co | Alternative to .com |
| .xyz | Modern, creative projects |

## Pricing Context

- Standard domains: ~$10–15/year
- Premium TLDs (.io, .ai): ~$30–50/year
- Taken domains: Market price if listed
- Premium domains: $hundreds to $thousands

## Tips

1. **Act fast**: Good domains get taken quickly
2. **Register variations**: Get .com and .io to protect the brand
3. **Check social media**: Make sure @username is available too
4. **Check trademarks**: Ensure no legal conflicts
5. **Say it out loud**: Test if it's easy to pronounce
6. **Think long-term**: Will it still make sense in 5 years?
