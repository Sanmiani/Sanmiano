## Key Workflow

1. Identify the communication type
2. Load the matching guideline file from `examples/`
3. Follow the specific instructions for formatting, tone, and content structure
4. If the communication type doesn't fit these categories, ask for clarification about the desired format

## 3P Update Format (Most Common)

```
**Progress** — What was accomplished since last update
**Plans** — What will be done before next update  
**Problems** — What is blocked or needs help

Keep each section to 3-5 bullet points max.
```

## General Communication Principles

- Lead with the most important information
- Use plain language — no corporate jargon
- Be specific with dates, numbers, and owners
- Keep it scannable: headers, bullets, short paragraphs
- State what action (if any) is needed from the reader
